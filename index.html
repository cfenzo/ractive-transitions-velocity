Ractive.js velocity transition plugin

download: ractive-transitions-velocity.js more plugins at docs.ractivejs.org/latest/plugins

This plugin let you use Velocity.js to make Ractive.js transitions, with full support for all of Velocity.js' properties and options (except complete, which is used by the plugin itself)

And if you add the Velocity.js UI Pack aling with Velocity.js (optional, but recommended), all the packaged effects will be readily available as transitions too.
